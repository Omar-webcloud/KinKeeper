# KinKeeper

KinKeeper is a modern personal relationship management web app designed to help you nurture and tend to your meaningful connections. Think of it as a personal shelf for your friends, ensuring you never lose touch with the people who matter most.

## 🚀 Technologies Used
- Next.js 15 (App Router)
- React 19
- Tailwind CSS
- Recharts
- Lucide React Icons
- LocalStorage state management

## ✨ Key Features
1. **Dynamic Friend Profiles**: Browse, track, and manage connections logically with distinct goals, tags, and dynamic status identifiers. Detailed profiles allow you to act upon connections immediately.
2. **Timeline Logging & Tracking**: Seamlessly log Interactions (Calls, Texts, Videos) instantly. They render dynamically onto a fully filterable holistic interaction Timeline.
3. **Friendship Analytics**: Data visualization using native Pie charts from Recharts allows a clear glimpse into the composition and interaction habits to build relationships.

Enjoy keeping your relations close!
