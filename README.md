# KinKeeper

KinKeeper is a modern personal relationship management web app 

### Live Link: https://kin-keeper.vercel.app/ 

## Technologies Used
- Next.js 15 (App Router)
- Tailwind CSS + DaisyUI    
- Recharts
- Lucide React Icons

## Key Features
1. **Dynamic Friend Profiles**
2. **Timeline Logging & Tracking**
3. **Friendship Analytics**
